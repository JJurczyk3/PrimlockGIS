# Primelock GIS

Primelock GIS is a from-scratch GIS project written in Python for an Advanced GIS coursework assignment.  
The goal is to build a small educational GIS system without relying on external GIS libraries, while implementing the core algorithms manually.

The system is designed to work with sampled geographic coordinate data and supports grid-based interpolation, convex-hull TIN generation, contour generation, and topology construction.


## Getting Started

### Requirements:
- Python 3.13+
- No external GIS libraries
